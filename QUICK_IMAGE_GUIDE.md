## Steps to Change/Upload Images

### Step 1: Upload New Portfolio Images
1. Go to **Upload Portfolio** tab
2. Enter image title (e.g., "Wedding Ceremony")
3. Select category (Wedding, Events, Baby, etc.)
4. **Drag drop** image file OR click to browse
5. See instant preview
6. Click **Upload Image** button
7. Success message appears ✓

### Step 2: Manage Existing Images
- View all images in **Manage Images** tab
- See thumbnail, title, category
- Click **Delete Image** to remove any image
- Changes apply instantly

### Step 3: Add/Change Services
- Click **Manage Services** tab
- Add new service with title, description, image URL
- Delete old services anytime
- Services appear on Services page automatically

## Image Requirements

✓ Format: JPG, PNG, GIF  
✓ Size: Under 5MB  
✓ Recommended: 600x500px for best quality  
✓ Title: Required (helps organize images)  
✓ Category: Pick from dropdown

## Drag & Drop Upload

Just drag your image from File Explorer directly into the upload zone:
1. No need to click browse button
2. See preview instantly
3. Upload immediately
4. Super fast!

---

## Common Tasks

### Change Portfolio Images
1. Delete old images → Manage Images tab
2. Upload new images → Upload Portfolio tab
3. Done! Changes show everywhere

### Update Services
1. Delete old service → Manage Services tab
2. Add new service with new image URL
3. Appears on Services page instantly

### Remove Images
1. Manage Images tab
2. Find image
3. Click Delete
4. Confirm
